# Golden Hour Café

A modern and responsive café website built with HTML, CSS, Bootstrap 5 and JavaScript.

## About the Project

Golden Hour Café is a fictional coffee shop website created as a front-end web development project. The goal was to design a clean, modern and responsive website that provides visitors with information about the café, its menu and contact details.

The website focuses on user-friendly navigation, responsive layouts and a warm visual identity inspired by the colors of coffee and golden hour sunsets.

## Features

* Responsive design for desktop, tablet and mobile devices
* Modern navigation bar
* Interactive hero sections
* Featured drinks section
* About page with café story and values
* Menu page with categorized products

  * Hot Drinks
  * Cold Drinks
  * Desserts
* Contact page with contact information and message form
* Scroll-to-top button
* Hover animations and interactive UI elements
* Bootstrap Icons integration

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5.3
* Bootstrap Icons
* JavaScript (Vanilla JS)
* Google Fonts (Poppins)
